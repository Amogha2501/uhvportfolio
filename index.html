My Work

The Medical Health Bot is an AI-driven chatbot designed to provide users with instant medical guidance, symptom assessment, and basic health-related information. It acts as a virtual assistant that interacts with users through natural language, helping bridge the gap between patients and healthcare resources—especially in situations where immediate consultation with a healthcare provider is not available. This bot was developed with the goal of increasing accessibility to preliminary healthcare information and reducing unnecessary hospital visits for minor or common symptoms. It is not a replacement for professional diagnosis but rather a supportive tool to help users make informed decisions.

  • Role: My Role: I was responsible for designing, developing, and testing the medical health bot, ensuring accurate responses.
  • Technologies Used:Natural Language Processing (NLP),C
  • Key Features/Learnings:

Harvest Guard is concluded to be a comprehensive solution addressing water scarcity in agriculture by optimizing rooftop rainwater use through technology and user-centric design. It empowers farmers, reduces reliance on traditional sources, promotes sustainability, enhances decision-making and resilience, and contributes to environmental conservation and farmer well-being. In summary, Harvest Guard is an app-based project designed to modernize and improve rainwater harvesting practices for farmers by providing them with digital tools and information to better manage water resources and improve agricultural outcomes.

This project is a complete UI/UX design prototype for a food catering service, created entirely using Figma. The goal was to conceptualize and design a visually engaging, user-friendly web application interface that allows users to explore menu options, customize catering packages, and submit event bookings. The design emphasizes clarity, accessibility, and responsiveness to provide a seamless experience across different devices. The project includes multiple screens such as: Home Page: A welcoming introduction with featured dishes and navigation. Menu Page: Categorized food sections (e.g., Starters, Main Course, Desserts) with card layouts. Order Page: A flow for selecting items, quantities, and scheduling delivery. Contact/Booking Page: A form for event catering inquiries. All screens were designed with a focus on real-world usability and a clean visual hierarchy, using consistent color themes, typography, and spacing.

  • Role: I led the UI/UX design process in Figma, from wireframing to high-fidelity prototyping. This included planning user flows, structuring layouts using auto-layout and building an interactive prototype to simulate user interaction.
